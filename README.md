🚍 SmartBus Booking Application on AWS

A production-style Bus Booking Application deployed on Amazon Web Services using a multi-tier architecture. The application is hosted on Amazon EC2, uses Amazon RDS MySQL for persistent storage, and is integrated with an Application Load Balancer for scalable and reliable traffic distribution.

Key Features:
• Online bus ticket booking
• Dynamic booking storage in Amazon RDS
• Flask-based backend application
• Application Load Balancer integration
• Secure connectivity using Security Groups
• Production-ready deployment architecture

AWS Services Used:
• Amazon EC2
• Amazon RDS (MySQL)
• Application Load Balancer (ALB)
• Security Groups

Tech Stack:
Python, Flask, PyMySQL, HTML, CSS, AWS
